# Penggunaan Aplikasi & Troubleshooting

## Cara Menggunakan Aplikasi Tes

### 1. Buka File .seb

1. Pastikan komputer terhubung ke **internet**
2. Buka folder **Desktop** (atau tempat Anda menyimpan file `p3musu.seb`)
3. **Klik dua kali** file `p3musu.seb`
4. SEB akan terbuka dan menampilkan halaman **form login aplikasi CBT**

![Form Login CBT](/images/instalasi-seb/seb-quiz.svg)

### 2. Saat Tes Berlangsung

Saat SEB aktif:
- **Hanya aplikasi CBT yang bisa diakses** — semua aplikasi lain terkunci
- Tombol **Alt+Tab**, **Windows key**, dan **Task Switcher** dinonaktifkan
- **Copy-paste**, **screenshot**, dan **print screen** diblokir
- Kursor dan navigasi terbatas pada area tes
- Gunakan **smartphone** terpisah untuk komunikasi dengan panitia via **Zoom** atau **WhatsApp**

### 3. Keluar dari SEB

- Setelah tes selesai, SEB akan otomatis tertutup
- Jika tidak: tekan **Ctrl+Alt+Q**
- Atau klik ikon **Quit SEB** di pojok kanan bawah layar
