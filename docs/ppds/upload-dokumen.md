# Upload Dokumen

Sistem PPDS tidak memiliki halaman "Upload Dokumen" terpisah. Setiap dokumen persyaratan diupload langsung melalui tab yang sesuai di dalam **Form Biodata**.

## Lokasi Upload Dokumen

Berikut adalah tabel referensi yang menunjukkan di mana setiap dokumen harus diupload:

| Dokumen | Upload di Tab | Keterangan |
|---------|--------------|-----------|
| Foto Terbaru | Data Diri | JPG, max 1MB, 2500x1600px |
| Ijazah S1 + Profesi | Pendidikan Kedokteran | JPG, max 1MB per file |
| Sertifikat Non Formal | Pendidikan Non Formal | JPG, max 1MB |
| Cover Karya Tulis | Karya Tulis | JPG, max 1MB |
| Halaman Pertama Karya Tulis | Karya Tulis | JPG, max 1MB |
| Sertifikat Prestasi | Prestasi | JPG, max 1MB |
| Bukti Pembayaran | Menu Pembayaran (terpisah) | JPG/PNG |

## Format File

Semua dokumen yang diupload **wajib dalam format JPG** dengan ketentuan:

- Ukuran maksimal: **1 MB per file**
- Resolusi maksimal: **2500 x 1600 piksel**

Pengecualian: Bukti pembayaran dapat diupload dalam format **JPG atau PNG**.

## Status Upload

Setiap dokumen akan memiliki status:

| Status | Keterangan |
|--------|-----------|
| Belum Upload | Dokumen belum diupload |
| Terupload | Dokumen sudah diupload |
| Diverifikasi | Dokumen sedang diperiksa admin |
| Diterima | Dokumen sesuai ketentuan |
| Ditolak | Dokumen tidak sesuai, perlu upload ulang |

## Panduan Detail

Untuk panduan lengkap mengenai tata cara mengisi form biodata dan mengupload dokumen, silakan buka halaman [Form Biodata](/ppds/form-biodata).
